Página de acceso a documentación y operación de las Apis Usig bajo especificación OpenAPI 3 (OAS). La especificación OAS define un estándar que permite el entendimiento de las capacidades de los servicios sin necesidad de acceso a los códigos fuente. Esta página detalla los tipos de parámetros implicados en el uso de las Apis asi como también la posibilidad de crear tests para consumirlas.
