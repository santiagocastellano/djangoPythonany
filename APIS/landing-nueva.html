Aplicaciones

  • Mapa Interactivo de Buenos Aires

    Mapa Interactivo Oficial de la Ciudad de Buenos Aires. Permite buscar direcciones y lugares de interés en la Ciudad de Buenos Aires y su área metropolitana, buscar recorridos en transporte público, auto, bici o pie y consultar mapas temáticos con información geográfica oficial

  • BA Cómo Llego

    Aplicación mobile especialmente diseñada para posibilitar el acceso a la búsqueda de recorridos del Mapa Interactivo de Buenos Aires a través de dispositivos móviles con acceso a Internet.

    • BA Cómo Llego (web)
    • BA Cómo Llego (Android)
    • BA Cómo Llego (iOS)
  • Normalizador de Bases de Direcciones

    Una solución amigable y efectiva para la normalización automática de direcciones en conjuntos de datos. Incluye las herramientas necesarias para corregir interactivamente los registros erróneos y la posibilidad de geocodificar de forma rápida, sencilla y precisa.

Servicios Web

  • Normalización de direcciones en AMBA

    Permite buscar, normalizar y geocodificar direcciones tanto dentro de CABA como en el área metropolitana.

  • Consulta de datos útiles

    Consulta de Barrio, Comuna, Comisaría, Área Hospitalaria, Región Sanitaria, Código Postal, Código de Planeamiento Urbano y Distrito Escolar para un lugar determinado.

  • Búsqueda de lugares de interés

    API para la búsqueda de lugares de interés en CABA y su área metropolitana.

  • Geocodificación de direcciones en CABA

    Permite geocodificar direcciones dentro de CABA utilizando un algoritmo de alta precisión de USIG.

  • Consulta de datos catastrales de CABA

    Provee una serie de métodos que permiten obtener información de las parcelas del catastro de la Ciudad de Buenos Aires..

Componentes para Desarrolladores

  • USIG-JS

    Biblioteca de componentes Javascript reutilizables para aplicaciones web que requieren interactuar con servicios de USIG.
    Documentación - Demos - Tests - Ejemplos

  • Normalizador de direcciones JS - CABA

    Implementación Javascript de un normalizador de direcciones que utiliza el callejero de USIG para transformar un string en una direccion normalizada. Requiere jQuery.
    Documentación - Demo - Tests - Demo del Buscador

  • Normalizador de direcciones Python - AMBA

    Componente Python para normalizar direcciones del AMBA (conurbano y CABA).
